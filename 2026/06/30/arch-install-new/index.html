准备磁盘

查看要安装系统的磁盘设备

重要! 确保系统安装在正确的位置, 避免抹除掉重要数据

对于不同类型的设备, 设备路径和分区路径有所不同, 这里举一个简单的例子, 实际设备名会有细微的不同:

  • m.2 接口的 nvme 协议设备: /dev/nvme0n1
    • boot partition: /dev/nvme0n1p1
    • swap partition: /dev/nvme0n1p2
    • root partition: /dev/nvme0n1p3
  • sata 接口或 usb 接口设备: /dev/sda
    • boot partition: /dev/sda1
    • swap partition: /dev/sda2
    • root partition: /dev/sda3

安装必要软件包

根据具体需求安装软件包:

  • 基础包
    • base: 基础系统工具
    • base-devel: 编译工具, Arch AUR 依赖
    • linux: 系统内核
    • linux-firmware: 常见固件
  • CPU 微码
    • intel-ucode: Intel CPU 微码
    • amd-ucode: AMD CPU 微码
  • 网络
    • networkmanager: 目前为止功能最全的网络管理工具, 与 KDE 协同工作良好, 需要启动 NetworkManager.service, systemd-resolved.service 守护进程
    • modemmanager: 移动宽带支持, 需要启动 ModemManager.service 守护进程
    • usb_modeswitch: 移动宽带支持
    • ppp: PPPoE / DSL 支持
  • 蓝牙
    • bluez: 提供蓝牙协议栈, 需要启动 bluetooth.service 守护进程
    • bluez-utils: 蓝牙命令行工具
  • 性能调度
    • power-profiles-daemon
      • power-profiles-daemon: 调度工具, ASUS Laptop 用户可选, 需要启动 power-profiles-daemon.service 守护进程
      • python-gobject: 修复 power-profiles-daemon 报错
      • asusctl: ASUS Laptop 管理配置工具, 需要添加 ASUS Linux 源, ASUS Laptop 用户可选
      • rog-control-center: asusctl 的 GUI 工具, 需要添加 ASUS Linux 源
    • tlp
      • tlp: 新一代调度工具, ThinkPad 用户可选, 需要启动 tlp.service 守护进程
      • tlp-pd: tlp 与 KDE 集成依赖
      • tlp-rdw: tlp 与 无线设备集成依赖, 需要启动 NetworkManager-dispatcher.service 守护进程
  • 显卡与视频
    • Intel
      • mesa: 显卡驱动
      • lib32-mesa: 显卡驱动 32 位支持
      • vulkan-intel: 显卡驱动 Vulkan 支持
      • lib32-vulkan-intel: 显卡驱动 Vulkan 32 位支持
      • intel-media-driver: Intel VA-API 支持
    • AMD
      • mesa: 显卡驱动
      • lib32-mesa: 显卡驱动 32 位支持
      • vulkan-radeon: 显卡驱动 Vulkan 支持
      • lib32-vulkan-radeon: 显卡驱动 Vulkan 32 位支持
    • NVIDIA
      • nvidia-open: 显卡驱动
      • nvidia-utils: VDPAU, NVENC, NVDEC, Vulkan 支持
      • lib32-nvidia-utilsVDPAU, NVENC, NVDEC, Vulkan 32 位支持
      • libva-nvidia-driver: 基于 CUDA NVDEC 的 VA-API 后端
    • Vulkan
      • vulkan-icd-loader: Vulkan 应用依赖
      • lib32-vulkan-icd-loader Vulkan 应用 32 位依赖
      • vulkan-tools: Vulkan 工具
    • 视频编解码
      • libvdpau-va-gl: 带 OpenGL/VAAPI 后端的 VDPAU 驱动
      • libva-utils: VA-API 工具
      • vdpauinfo: VDPAU 工具
  • 音频
    • sof-firmware: 某些板载声卡的固件
    • alsa-firmware: ALSA 固件
    • alsa-utils: ALSA 工具
    • pipewire: 全新多媒体框架, 需要以非 root 身份启动 pipewire.service 守护进程. 若安装 KDE, 则会自动启用该进程, 无需手动设置
    • lib32-pipewire pipewire 的 32 位支持
    • wireplumber: pipewire 会话管理器, 需要以非 root 身份启动 wireplumber.service 守护进程. 若安装 KDE, 则会自动启用该进程, 无需手动设置
    • pipewire-audio: pipewire 音频服务器基础
    • pipewire-alsa: pipewire ALSA 支持
    • pipewire-pulse: pipewire PluseAudio 支持, 需要以非 root 身份启动 pipewire-pulse.service 守护进程. 若安装 KDE, 则会自动启用该进程, 无需手动设置
    • pipewire-jack: pipewire JACK 支持
    • lib32-pipewire-jack pipewire JACK 32 位支持
  • 工具
    • sudo: 以非 root 用户身份执行特权命令
    • nano, vi, vim: 文本编辑器, vi 是全局默认编辑器, 若不选择安装则需要额外手动配置
    • man-db, man-pages, texinfo: 命令帮助手册
    • wget
    • git
    • openssh: 需要启动 sshd.service 守护进程
    • btop
    • fuse2
    • unzip
    • 7zip
    • tmux
    • tree
    • firewalld: 需要启动 firewalld.service 守护程序
    • libayatana-appindicator
  • 字体
    • noto-fonts
    • noto-fonts-cjk
    • noto-fonts-emoji
    • ttf-cascadia-code-nerd
  • 桌面环境
    • plasma-meta: 需要启动 plasmalogin.service 守护程序
    • konsole
    • dolphin
    • qt6-multimedia-ffmpeg
    • kwalletmanager
    • krdc
    • firefox
  • 输入法
    • fcitx5-im
    • fcitx5-chinese-addons
    • fcitx5-pinyin-zhwiki
