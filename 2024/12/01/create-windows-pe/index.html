在U盘创建Windows PE

发表于2024-12-01|更新于2025-11-08|杂项
|浏览量:

参考链接:

https://learn.microsoft.com/zh-cn/windows-hardware/manufacture/desktop/winpe–use-a-single-usb-key-for-winpe-and-a-wim-file—wim?view=windows-11#option-1-create-a-multiple-partition-usb-drive

准备工作

  1. 下载 Windows ADK 10.1.26100.2454 (2024 年 12 月) 和 适用于 Windows ADK 10.1.26100.2454 的 Windows PE 附加组件 (2024 年 12 月).

    请下载最新版 Windows ADK, 这里只是当时的最新版本, 见 https://learn.microsoft.com/zh-cn/windows-hardware/get-started/adk-install.

  2. 以管理员身份运行 部署和映像工具环境 (位置: 开始菜单 -> Windows Kits), 并执行相关命令, 以下命令均在 部署和映像工具环境 中运行.

具体步骤

  1. 运行 copype 以创建 Windows PE 文件的工作副本.
  1. 挂载 Windows PE 副本为 wim 镜像, 以便自定义 Windows PE.
  1. 将所需要的包添加到 Windows PE 中, 由于本人只需要通过该 Windows PE 安装 Windows 10 / 11 操作系统, 这里只添加相关组件.

    从 Windows PE 启动 Windows 11 安装程序, 需添加 WinPE-WMI 和 WinPE-SecureStartup 可选组件, 参考 https://learn.microsoft.com/zh-cn/windows-hardware/manufacture/desktop/winpe-create-usb-bootable-drive?view=windows-11#common-customizations.
    在 Windows PE 环境中挂在 ISO 镜像可以通过 Mount-DiskImage 命令实现, 因此还需要添加 PowerShell 相关组件, 组件内容参考 https://learn.microsoft.com/zh-cn/windows-hardware/manufacture/desktop/winpe-adding-powershell-support-to-windows-pe?view=windows-11.
    关于语言包: 需要安装基础语言包 xx-xx\lp.cab ,以及字体包 WinPE-FontSupport-xx-xx.cab (不一定存在); 对于每个可选组件, 需要添加语言无关的包和一些与其关联的特定的语言包, 比如 WinPE-WMI.cab 和 xx-xx\WinPE-WMI.cab, 参考 https://learn.microsoft.com/zh-cn/windows-hardware/manufacture/desktop/winpe-add-packages–optional-components-reference?view=windows-11.

  1. 检查已添加的组件.
  1. 将 Windows PE 区域设置更改为你要使用的语言, 这里修改为 ZH-CN.
  1. 移除 Windows PE 镜像挂载, 保存修改操作.
  1. 使用 Diskpart 重新格式化驱动器, 并为 WinPE 和映像创建两个新分区.

    这里将 U 盘分为两个分区, 一个分区存放 Windows PE 本体, 另外一个分区存放 ISO 镜像以及其他文件, 注意请将 X 修改为 U 盘的序列号, 通过 list disk 命令查询, 参考 https://learn.microsoft.com/zh-cn/windows-hardware/manufacture/desktop/winpe–use-a-single-usb-key-for-winpe-and-a-wim-file—wim?view=windows-11#option-1-create-a-multiple-partition-usb-drive.

  1. 将 WinPE 文件复制到 WinPE 分区
