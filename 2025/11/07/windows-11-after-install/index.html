Windows 11 全新安装记录

发表于2025-11-07|更新于2026-01-07|杂项
|浏览量:

适用于 Windows 11 25H2, 其他版本未作测试, 并长期不定期更新.
测试平台: ASUS TUFGaming A16 2024 FA607PV, 其他平台可能存在差异.

具体安装过程

  1. 访问 uupdump, 选择 Windows 11 最新版本创建下载包, 注意选择 下载, 添加虚拟升级版本并转换为 ISO 镜像文件.
  2. 解压下载包, 修改 ConvertConfig.ini 中 CustomList =0 为 CustomList =1; 修改 CustomAppsList.txt 中去掉 Microsoft.WindowsNotepad_8wekyb3d8bbwe, Microsoft.WindowsCalculator_8wekyb3d8bbwe, Microsoft.ScreenSketch_8wekyb3d8bbwe, 以及 标题为### Media Codecs / Client non-N editions, Team edition 下的所有列表项的 # 注释, 然后以管理员身份运行 uup_download_windows.cmd, 等待创建 ISO 镜像完成.
  3. 使用多种方式载入 ISO 镜像, 安装 Windows 11, 时间和货币格式选择 简体中文(新加坡), 输入密钥 XGVPP-NMH47-7TTHJ-W3FW7-8HV2C 以选择企业版, 安装完成后重启不要引导 Windows 11 的硬盘, 通过再次载入 ISO 镜像或其他方式关闭计算机, 拔掉加载 ISO 镜像使用的外接存储设备.
  4. 再次开机后进入 OOBE 页面, 进行 Windows 的配置, 此时伴随着 Windows 更新会自动安装驱动程序.
  5. OOBE 完成后进行一些系统配置和应用配置, 配置项详见本文 系统设置 和 应用设置 标题.
